# AI Agent Operating Agreement

Defines permissions, boundaries, and protocols for agents with codebase access.

- Allowed actions: create/edit code, docs, tests
- Restricted areas: secrets, production data
- Escalation: PR review required for risky changes

## Ingestion Order
1. Read PBC
2. Read this Operating Agreement
3. Read relevant `skills/*.skill.md` for the current task
4. Follow links to canonical docs (ADRs, policies, playbooks)
